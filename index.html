DATA SOURCES
OpenCelliD · Mobile cells
Open, collaborative database with millions of mobile cells worldwide. The query uses the selected country's MCC and shows operators detected by MNC.
Nominatim · Geocoding
OpenStreetMap geocoding service. It converts a text address into lat/lon coordinates to query nearby cells.
OpenStreetMap · Base map
Free collaborative map of the world. It is used as the base layer to visualize cells and the searched place.
Signal index · Local calculation
The index combines: number of cells in the radius, operator diversity, LTE/NR (4G/5G) presence, and distance to the nearest cell. It does not measure real signal power.
Global references · Additional validation
nPerf, Ookla, Opensignal, and GSMA are used as international references. They complement cell analysis but do not replace field measurements.
OpenCelliD data is crowdsourced and may not reflect all real infrastructure · For official coverage, check the corresponding operator maps · License: OpenCelliD CC BY-SA 4.0 · OSM ODbL
